# 🐳 Docker Demo

This folder contains a **dummy Dockerfile** created for demonstration purposes.

## 📌 Purpose

The goal of this Docker setup is to showcase basic containerization knowledge without affecting the actual backend or deployment.

## ⚠️ Important Note

* This does **NOT** run the real backend server
* It is **not used in production**
* It does **not expose any ports or services**

## 🚀 How to Run

```bash
docker build -t docker-demo .
docker run docker-demo
```

## 🧾 Output

```
This is a dummy Docker container for showcase. No server is running.
```

## 📁 Files

* `Dockerfile` → Minimal container using Alpine Linux
* `README.md` → Documentation for this demo setup
