# Demo Data for OpsSync

This folder contains mock data to demonstrate the incident lifecycle and system workflow.

## Included Data

- engineers.json → Simulates available incident engineers
- incidents.json → Represents incidents in different stages
- logs.json → Shows actions performed during incident resolution

## Purpose

This data is designed to:
- Demonstrate incident assignment
- Show lifecycle transitions (OPEN → RESOLVED)
- Highlight engineer activity during incident handling

## Note

This demo data does not interact with the actual backend or database.
It is purely for demonstration and testing purposes.
